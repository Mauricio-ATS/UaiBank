**UaiBank Project**
Project based on create a bank that has functions using **terminal lines** and save the data in a file **CSV**.
main menu in **code.c**:

MENU UAIBANK

1 - Insert New User

2 - Insert More then One User

3 - Get info about a User

4 - Pament between Users

5 - Remove User

All information is present in **clientes.csv** (ID, NOME, IDADE, SALDO).
